M-Q Project

A media query project involves using CSS to make a website look good an different devices, like phones, tablets, and computers. By using media queries, you can adjust the layout, font size, and other design elements to ensure the website looks great and is easy to use, no matter what devices someone is using to view it. This helps provide a better experience for users and ensures your website is accessible across a variety of devices and screen sizes.

Sass Project

SASS is preprocessor to write CSS in a more organized and efficient way. SASS allows you to use features like variables, nesting, mixins and inheritance to streamline your CSS code and make it easier to maintain and scale. with SASS you can write stylesheets faster and with less repetition, which helps speed up development and keep your codebase clean and organized. In simple terms, it's like supercharging your CSS workflow to make styling websites or web app more efficent and powerful.
